Soal 1: Mengelola Daftar Siswa 
	    Buatlah sebuah program untuk mengelola daftar nama Mahasiswa
	    yang mengikuti ujian. Program ini harus mampu:

	    Menambahkan nama Mahasiswa ke dalam list.
	    Menghapus nama Mahasiswa dari list jika Mahasiswa tersebut mengundurkan diri.
	    Menampilkan daftar nama Mahasiswa yang mengikuti ujian.
	
        Petunjuk: Gunakan operasi append(), remove(), dan print().


Soal 2: Mencari Nilai Tertinggi dan Terendah
	    Buat program yang menerima input nilai ujian dari sejumlah Mahasiswa
	    dan menyimpan nilai-nilai tersebut dalam list. Program ini harus:

        Menampilkan nilai tertinggi.
        Menampilkan nilai terendah.
        Menampilkan nilai rata-rata.
        Petunjuk: Gunakan fungsi max(), min(), dan sum().

        Contoh Input dan Output:

        Input: nilai = [85, 90, 78, 92, 88]
        Output:
        Nilai Tertinggi: 92
        Nilai Terendah: 78
        Rata-rata Nilai: 86.6

Soal 3: Membalik Urutan List
        Buat program yang menerima input sebuah list dan membalikkan urutan
        elemen dalam list tersebut. Program ini harus:

        Membalikkan list tanpa menggunakan metode reverse().
        Petunjuk: Kamu bisa menggunakan slicing atau perulangan untuk membalikkan list.

        Contoh Input dan Output:

        Input: list_angka = [1, 2, 3, 4, 5]
        Output: [5, 4, 3, 2, 1]
        
Soal 4: Menyaring Angka Ganjil dan Genap
        Buat program yang menerima input sebuah list berisi angka dan memisahkan
        angka ganjil dan genap ke dalam dua list terpisah.

        List pertama harus berisi angka genap.
        List kedua harus berisi angka ganjil.
        Petunjuk: Gunakan operasi modulus (%) untuk memeriksa angka ganjil dan genap.

        Contoh Input dan Output:

        Input: angka = [1, 2, 3, 4, 5, 6, 7, 8]
        Output:
        List Genap: [2, 4, 6, 8]
        List Ganjil: [1, 3, 5, 7]
        
Soal 5: Daftar Belanja
        Buat program untuk mengelola daftar belanja. Program ini harus:

        Menambahkan item ke daftar belanja.
        Menghapus item yang sudah dibeli.
        Menampilkan daftar belanja saat ini.
        Petunjuk: Gunakan metode append(), remove(), dan print().

        Contoh Input dan Output:

        Input: Menambahkan "Apel", "Pisang", "Susu".
        Output: ['Apel', 'Pisang', 'Susu']
        Input: Menghapus "Pisang".
        Output: ['Apel', 'Susu']
	
Soal 6: Mencari Elemen di List
        Buat program yang menerima sebuah list dan mencari apakah sebuah nilai
        ada di dalam list. Jika ada, tampilkan indeksnya, jika tidak, tampilkan
        pesan bahwa nilai tersebut tidak ditemukan.

        Petunjuk: Gunakan metode index() atau operator i

        Contoh Input dan Output:

        Input: nilai = [50, 75, 60, 85, 90]
        
        Output: Indeks nilai 60 adalah 2.
	
Soal 7: Menyatukan Dua List
        Buat program yang menggabungkan dua list menjadi satu list.		
        Program ini harus menerima dua list sebagai input
        dan menggabungkannya dalam urutan yang benar.

        Petunjuk: Gunakan operator + atau metode extend().

        Contoh Input dan Output:

        Input: list1 = [1, 2, 3] dan list2 = [4, 5, 6]
        Output: [1, 2, 3, 4, 5, 6]
        

Soal 8: Menyusun List dalam Urutan Terbalik
        Buat program yang menerima sebuah list dan menyusunnya dalam
        urutan terbalik. Program ini harus menampilkan list yang sudah diurutkan secara terbalik.

        Petunjuk: Gunakan metode sort(reverse=True) atau sorted().

        Contoh Input dan Output:

        Input: list_angka = [1, 3, 2, 5, 4]
        Output: [5, 4, 3, 2, 1]


soal 9: Pengelolaan Nilai Mahasiswa
        Sebuah universitas ingin membuat program sederhana untuk mengelola nilai mahasiswa. Program ini harus memiliki fitur berikut:
        Meminta pengguna memasukkan nama-nama mahasiswa dan nilai mereka (dalam bentuk list).
        Menghitung rata-rata nilai seluruh mahasiswa.
        Menampilkan daftar mahasiswa yang mendapatkan nilai di atas rata-rata.
        Menampilkan daftar mahasiswa yang mendapatkan nilai di bawah rata-rata.
        
        Kebutuhan Program:

        Gunakan list untuk menyimpan nama dan nilai mahasiswa.
        Gunakan loop untuk menghitung rata-rata dan menentukan hasil akhir.
